Push code ke GitHub tanpa kata sandi

Cara push ke Github tanpa password, 23 Mar 2019

SSH (Secure SHell) merupakan perangkat lunak yang dapat digunakan untuk mentransfer data antar komputer secara terenkripsi. Sehingga kita dapat melakukan transfer data pada git dengan menggunakan ssh atau git over ssh.

Untuk menggunakan ssh kita perlu melakukan otentikasi pada komputer tujuan, salah satu metode otentikasi adalah dengan menggunakan private-public key pair. Dengan menggunakan metode ini maka proses transfer data dapat dilakukan tanpa perlu mengetikkan kata sandi setiap kali proses transfer data dilakukan.

Fitur dari ssh inilah yang kita gunakan untuk push codekita ke github tanpa kata sandi. Caranya gampang gaes langsung disimak saja.

  1. Persiapkan public key ssh, bila anda belum punya maka buka terminal dan ketik perintah berikut:
    $ ssh keygent -t rsa
    Setiap pertanyaan yang muncul tekan tombol enter saja

  2. Copy isi file dari ~/.ssh/id_rsa.pub

  3. Klik tautan berikut.
    Kemudian login jika anda belum melakukannya.

  4. Isikan name key anda pada kolom Title>, misal: ubuntu laptop key.
    Kemudian paste isi dari file id_rsa.pubyang tadi di copy ke kolom Key.

  5. Pastikan remote url yang anda sudah mneggunakan ssh. Buka terminal dan ketik perintah berikut:
    $ git remote set-url origin git@github.com:USERNAME/REPOSITORY.git
    Anda rubah USERNAME/REPOSITORY diatas dengan username dan nama repository anda
    6. .

Voila, anda sudah dapat melakukan sinkronisasi dengan github anda tanpa kata sandi.

Blog Body Layout

Konten Layout dari blog ini dalam HTML & CSS, 23 Maret 2019

Salah satu bagian dari proses pembuatan website adalah men-design layout sebagai kerangka website tersebut. Langsung saja gaes, layout website ini dalam css.

Flow HTML adalah cara browser me-render dari atas kebawah. Dan Elemen-elemen HTML dipandang sebagai sebuah box. Ide dari layout ini adalah membuat 2 kontainer utama yang akan ditampilkan berdampingan, kemudian dari masing-masing kontainer tersebut akan di buat beberapa child element sebagai kontent dari website.

Pertama-tama buat 2 kontainer utama, dan tentukan width dari masing elemen serta tambahkan background-color dan height nya agar kontainer dapat terlihat. Berikut merupakan tampilan dari kontainer tersebut dengan nilai untuk masing-masing property, width: 75% dan 25%, background-color: green dan black, height: 400px.

Kemudian tambahkan properti float: left pada kedua kontainer utama agar tampilannya sejajar. Selanjutnya buat child element pada kedua kontainer tersebut sebagai kontainer untuk kontent. Dan Voila, jadilah kerangka layout untuk blog kita.
